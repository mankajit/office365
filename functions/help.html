ADBS Excel Custom Functions

=ADBS.ADTOBS(A1) — Convert the AD date in cell A1 to BS (YYYY-MM-DD).

=ADBS.ADTOBS(2025,8,12) — Convert specific AD Y,M,D to BS.

=ADBS.ADTOBS_LONG(A1) — Return long format (e.g., '17 Poush 2079').

Note: Demo range uses BS data for ~2075–2085. Extend bsMonthData in converter.js for wider coverage.
